Block VS Inline elements

Block elements - is a container to group other elements. It contains a block or inline element.

By default, it begins on new lines. e.g div, section, nav, p, article, etc...

Inline Elements - is a container for text. It contains other inline elements. It can start anywhere in a line e.g <span>, <em> etc.

The HTML <body > -- Element represents the content of an HTML document. It contains the content to be displayed and all the markup code for the entire page.

The HTML <p> -- element represents a paragraph. It is used to tell browser we want a paragraphs and is a text block element

Syntax

  • Body - <body> ... </body >
  • Paragraph - <p> ... </p>
