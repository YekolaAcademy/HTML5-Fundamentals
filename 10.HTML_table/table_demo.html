HTML Table Syntax

  • The < table >tag define and display the HTML tables.
  • The < tr > tag define and display table row
  • The < th > tag define and display table header
  • The < td > tag define and display table data

Symantic Table Syntax

  • <thead > - header that label the content in each column
  • <tbody > - body of the table
  • <tfoot > - describes the sum-up table information
  • <caption > - description of the table’s purpose and the information
